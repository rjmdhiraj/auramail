Voice Commands Guide

Navigation Commands

  • "read inbox" - Go to inbox
  • "compose email" - Start new email
  • "show settings" - Open settings
  • "go back" - Return to previous screen
  • "help" - Show this guide

Email Commands

  • "read email" - Read current email aloud
  • "reply" - Reply to email
  • "forward" - Forward email
  • "delete" - Delete email
  • "mark as spam" - Mark as spam

Composition Commands

  • "dictate recipient" - Voice input for recipient
  • "dictate subject" - Voice input for subject
  • "dictate message" - Voice input for message
  • "start dictating" - Begin voice input
  • "stop dictating" - End voice input
  • "send email" - Confirm and send
  • "confirm send" - Confirm sending
  • "save draft" - Save as draft with confirmation
  • "confirm save" - Confirm saving draft
  • "cancel" - Cancel action

System Commands

  • "repeat" - Repeat last announcement
  • "louder" - Increase volume
  • "quieter" - Decrease volume
  • "high contrast" - Toggle contrast mode
